# ORBIT — Adaptive Learning Platform

An adaptive AI system that evolves with your learning journey. From confusion to clarity.

## Features

- 🧠 **AI Career Recommendation** — Intelligent analysis of your strengths and interests
- 🗺️ **Structured Roadmaps** — Week-by-week learning plans with milestones
- 🔄 **Adaptive Feedback Loop** — Real-time behavior insights and adjustments
- 📊 **Clarity Index** — Visual progress tracking (0–100)
- 🔥 **Habit & Streak Tracking** — Stay consistent with daily task checklists
- 📈 **Insights & Reflection** — Engagement graphs and weekly self-review

## Tech Stack

- **React 18** + **Vite**
- **React Router** for client-side navigation
- **Chart.js** for data visualization
- **CSS** with custom design system (dark theme, glassmorphism)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Pages

| Route | Description |
|---|---|
| `/` | Landing page |
| `/login` | Login / Signup |
| `/dashboard` | Home dashboard with field selection |
| `/stream-selection` | Choose a specialization stream |
| `/stream-dashboard` | Roadmap, tasks, notes & feedback |
| `/insights` | Reflection, engagement graphs & skill bars |
